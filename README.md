# 🎤 ラップDE老GO (Rap DE Rogo)

## 概要
**「人生、謳歌しよう。思い出をビートに乗せて。」**

「ラップDE老GO」は、シニア世代が歩んできた豊かな人生や思い出を、現代的なリズムである“ラップ”に乗せて表現する、新感覚のエンターテインメントWebアプリケーションです。

「ラップは若者のもの」「難しそう」といったハードルを、最新のAI技術（Google Gemini）が優しくサポートすることで解消。思い出のエピソードを入力するだけで、AIが韻を踏んだ歌詞を提案したり、言葉遊びのクイズを出してくれたりします。

演歌や童謡をアレンジした親しみやすいビートに乗り、お孫さんやご家族と一緒にワイワイ楽しみながら、世界に一つだけの「人生アルバム」を作ってみませんか？世代を超えたコミュニケーションの架け橋となることを目指しています。

## ✨ 主な特徴

### 1. 誰でもラッパーになれる「AI作詞サポート」
思い出のエピソード（例：「孫といった夏祭り」）を入力するだけで、AI（Google Gemini）がラップ調の歌詞を自動で提案します。作詞の経験がなくても、味わい深いリリックが完成します。

### 2. 楽しく脳トレ「韻（いん）あてクイズ」
言葉遊び感覚で楽しめる「韻あてクイズ」機能を搭載。単語を入力すると、AIが韻を踏む言葉と踏まない言葉を生成し、クイズ形式で出題します。楽しみながら頭の体操ができます。

### 3. シニア・ファーストなUIデザイン
*   **大きな文字とボタン**: 視認性を高め、押し間違いを防ぐデザイン。
*   **やさしい色使い**: 暖色系をベースにした、安心感のある配色。
*   **シンプルな操作**: 専門用語を避け、直感的に操作できる画面遷移。

### 4. 人生アルバム
作成したラップは、ジャケット画像と共に「人生アルバム」に保存されます。家族が集まった時に再生して、思い出話に花を咲かせることができます。

## 🛠 技術スタック

*   **Frontend**: React (TypeScript), Vite
*   **Styling**: Tailwind CSS
*   **AI**: Google Gemini API (@google/genai)
*   **Fonts**: Mochiy Pop One (タイトル), Noto Sans JP (本文)

## 🚀 使い方

1.  **ビートを選ぶ**: 演歌風、童謡風など、気分に合ったビートを選択。
2.  **歌詞を書く**: 思い出を入力してAIに手伝ってもらうか、韻クイズで言葉を探して入力。
3.  **録音する**: ビートに合わせて歌声を録音（デモ機能）。
4.  **アルバムに残す**: ジャケット写真と共にアルバムへ保存。
